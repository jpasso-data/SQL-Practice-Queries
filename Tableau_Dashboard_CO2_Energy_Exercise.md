# Tableau Dashboard: CO2 & Energy Data Visualization
**Course:** Google Data Analytics Certificate - Course 6  
**Author:** Joe Passo | jpasso-data  
**Date:** May 2026  

## Overview
Built a multi-sheet Tableau dashboard combining CO2 emissions 
and energy data across global regions from 1962–2012.

## Data Sources
- CO2 Data Cleaned
- Energy Data
- GDP Total
- Total Population

## Steps Completed
1. Connected 4 datasets in Tableau Public
2. Joined CO2 and Energy tables on Year and Country Name
3. Built Sheet 1: Average CO2 Per Capita bubble map by country/region
4. Built Sheet 2: CO2 Production by Region line graph (1962–2012)
5. Created Dashboard 1 combining both visualizations
6. Cleaned dashboard: removed redundant size legend, floated 
   color and region legends onto respective visualizations

## Key Concepts Practiced
- Multi-source data connections in Tableau
- Table joins with multiple join clauses
- Dashboard layout and design
- Floating vs. tiled legend placement
- Data storytelling through layered visualizations
